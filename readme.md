# JavaScript Mini Programs Collection

## Overview

This project is a collection of interactive web applications built using HTML, CSS, and JavaScript. It serves as a learning project to practice core JavaScript concepts, DOM manipulation, event handling, timers, random number generation, API integration, and responsive web design.

## Included Programs

### Counter

A simple counter that allows users to increment, decrement, and reset values.

### Number Guessing Game

A game where users attempt to guess a randomly generated number with feedback provided after each guess.

### Dice Roller

Simulates rolling dice and generates random outcomes.

### Temperature Converter

Converts temperature values between Celsius and Fahrenheit.

### Digital Clock

Displays the current time and updates automatically in real time.

### Stopwatch

Tracks elapsed time with start, stop, and reset functionality.

### Calculator

Performs basic arithmetic operations through a user-friendly interface.

### Rock Paper Scissors

A classic game where users play against the computer.

### Weather App

Fetches and displays real-time weather information for a specified city using the OpenWeather API.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* OpenWeather API

## Features

* Responsive user interface
* Interactive user experience
* Real-time updates
* API integration
* DOM manipulation
* Event-driven programming

## How to Run

1. Clone or download the project.
2. Open the project folder.
3. Launch `index.html` in a web browser.
4. Select any program from the home page to use it.
